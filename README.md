**Este repositório reúne projetos de análise de dados desenvolvidos em SQL**

Os exercícios práticos foram realizados dentro da plataforma de cursos da **DataCamp** e têm como objetivo praticar técnicas de consulta, exploração e manipulação de dados em bancos de dados relacionais.

| Projeto | Descrição | Habilidades |
|---|---|---|
| [Project - Factors that Fuel Student Performance](./Project%20-%20Factors%20that%20Fuel%20Student%20Performance) | Análise de dados acadêmicos para identificar como certas variáveis influenciam o desempenho dos alunos | SQL, agregações, CASE WHEN, window functions |
| [Project - Uncovering the World's Oldest Businesses](./Project%20-%20Uncovering%20the%20World%27s%20Oldest%20Businesses) | Exploração de dados históricos para identificação do negócio mais antigo por continente, análise de países sem registros e investigação de quais setores econômicos tendem a sobreviver por séculos | SQL, JOINs, agregações, union |
| [Project - Exploring Trends in American Baby Names](./Project%20-%20Exploring%20Trends%20in%20American%20Baby%20Names) | Identificação de tendências de popularidade em nomes de bebês nos Estados Unidos, analisando nomes mais frequentes ao longo do tempo, comparando padrões entre diferentes períodos etc. | SQL, agregações, window functions, CASE WHEN |
